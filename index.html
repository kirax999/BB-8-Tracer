Welcome to BB-8 Tracer.

BB-8 Tracer this alertnate application IOS to use BB-8 Bot by Sphero. BB-8 Tracer allow trace route for BB-8 and change color to LED.

BB-8 Tracer use official SDK provided by Sphero (https://github.com/orbotix/Sphero-iOS-SDK)
